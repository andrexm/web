Lunar

Lunar é uma biblioteca construída inteiramente com Javascript, que visa trazer soluções eficientes para alguns problemas comuns envolvendo a criação de interfaces dinâmicas e criação de determinados efeitos, que, com Javascript puro, seriam difíceis ou trabalhosos de se criar. Atualmente, o projeto possui uma boa quantidade de recursos, mas ainda há muito a se fazer.

Javascript, CSS
Singular

Esta é uma microframework escrita em PHP, cujo objetivo é ser uma estrutura limpa e suficiente para projetos que não necessitam de tantos recursos, porém contando com recursos úteis e eficientes, o que garante grande performance. Assim, o Singular é ideal para sites e projetos mais simples, que não necessitam de muito, além de incorporar o incrível sistema de templates do Laravel.

PHP, Blade
